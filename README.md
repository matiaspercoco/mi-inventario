#  Sistema de Gestión de Inventario Pro

Una aplicación web ligera, rápida y optimizada para dispositivos móviles diseñada para el control de stock personal o de pequeños negocios.

## 🚀 Características
- **Interfaz Mobile-First:** Diseñada para usarse cómodamente desde el celular con botones grandes y tarjetas táctiles.
- **Orden Automático:** Los productos se organizan alfabéticamente de forma automática.
- **Ajuste Rápido:** Botones de `+` y `-` para actualizar el stock sin entrar en menús.
- **Exportación:** Genera reportes en Excel (.xlsx) con formato profesional y columnas auto-ajustadas.
- **Buscador Inteligente:** Filtra productos en tiempo real.

## ⚠️ NOTA IMPORTANTE SOBRE LOS DATOS
> **Esta aplicación utiliza el almacenamiento local del navegador (LocalStorage).**
> 
> * **No guarda el progreso en la nube:** Los datos que cargues en un dispositivo (ej. tu PC) **no** se verán en otro (ej. tu celular).
> * **Riesgo de pérdida:** Si borras los datos de navegación, el historial o la caché del navegador, el inventario se eliminará permanentemente.
> * **Recomendación:** Se sugiere utilizar la función de **"Exportar Excel"** regularmente para tener un respaldo físico de tu información.

## 🛠️ Tecnologías utilizadas
* **HTML5** - Estructura semántica.
* **CSS3** - Diseño responsivo con variables y animaciones.
* **JavaScript (Vanilla)** - Lógica de inventario y manipulación del DOM.
* **SheetJS (XLSX)** - Librería externa para la generación de reportes en Excel.

## 📱 Cómo instalar en el celular
1. Abre el enlace de la app en tu navegador (Chrome o Safari).
2. Toca el menú de opciones (tres puntos en Android o el icono de compartir en iOS).
3. Selecciona **"Agregar a la pantalla de inicio"**.
4. ¡Listo! Ya tienes acceso directo como una aplicación nativa.
